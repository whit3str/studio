# **App Name**: PokéLearn Quest

## Core Features:

- Number Challenge: Quiz users by displaying a Pokémon image from the first 151, and asking for its National Pokédex number.
- Name Recall: Present a National Pokédex number (1-151) and challenge users to type the corresponding Pokémon's name.
- Kanto Region Study: Browse a comprehensive list of the first 151 Pokémon, displaying their images, names, and numbers for learning and review.
- AI Hint Tool: When a user is stuck on a quiz question, the AI provides a contextual hint about the Pokémon (e.g., type, common habitat, distinct feature) to guide them without revealing the direct answer.
- Progress Summary: After each quiz session, display the user's score, showing the number of correct and incorrect answers to track learning progress.

## Style Guidelines:

- Primary color: Energetic Orange-Red (#F45925). This bold hue evokes the adventure and iconic branding of the Pokémon universe, setting an exciting tone for the learning experience.
- Background color: Subtle Pale Peach (#FBFAF9). A heavily desaturated version of the primary hue, providing a bright, clean, and unobtrusive canvas that keeps the focus on the content without feeling stark.
- Accent color: Playful Raspberry (#EB4770). An analogous color chosen for its vibrancy and ability to draw attention, perfect for interactive elements like buttons, progress indicators, or visual feedback.
- Font for all text: 'Poppins' (sans-serif). Its clean, geometric forms provide excellent readability for a quiz app while maintaining a modern, friendly, and precise feel, suitable for displaying Pokémon names, numbers, and quiz prompts.
- Utilize classic 8-bit or stylized pixel art for Pokémon sprites and interface elements like Poké Balls and directional arrows, honoring the game's retro roots. Interactive icons should clearly convey their function.
- Adopt a clean, centered card-based layout for quiz questions and study mode, ensuring that each Pokémon or question is the focal point. Content should be easily digestible on mobile and desktop devices with clear input fields.
- Incorporate subtle bounce or shake animations for incorrect answers and a satisfying 'ping' or 'sparkle' for correct answers. Use smooth transitions when moving between quiz questions or sections to enhance user engagement.
